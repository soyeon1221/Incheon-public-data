## &#128310; 프로젝트 소개

App을 통해 지역과 년도별로 인천광역시 어린이사고 통계를 조회할 수 있습니다. 

Tailwind CSS를 사용하여 디자인하였습니다.

카카오 지도 API와 Chart 라이브러리를 활용하여 데이터를 시각적으로 표현합니다.
